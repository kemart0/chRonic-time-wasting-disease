1_CLASSIFIER

This directory contains the code used to train a machine learning classification model to detect prion positive vs prion negative brain images across different brain tissues at ~4x obj.

#################
RUNNING THIS CODE
#################

1 - Prepare training data. See DATA PREPARATION.
2 - Create conda environment for running "disease_classifier.ipynb". See CODE AND DEPENDENCIES.
3 - Run each code block of disease_classifier.ipynb to train and validate a classification model. 
4 - Resulting model is saved as "best_model.pth"

################
DATA PREPARATION
################

Sorted training and validation data is available on Alpine:
	/pl/active/courses/2026_summer/CSU_qCMB/chronic_time_wasting_disease/data
	
If you do not have access to Alpine:

Images were separated into prion "positive/" and "negative/" control directories manually. 

Two images in the positive group were at a higher objective and lacked representation in the negative group. These two images were removed: 
		GtDeer_treatment_septum_4x_31.tif
		GtDeer_treatment_septum_4x_32.tif

Data was then sorted into training (train) and validation (val) directories using the BASH script: split_images.sh

Data was then uploaded to Alpine prior to training the model.

#####################
